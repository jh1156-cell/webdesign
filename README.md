# 브랜드별 단가계산기

Streamlit으로 만든 거래처별 단가 계산 앱입니다.

## 주요 기능

- 거래처DB 시트의 `A3:A63`, `F3:F63` 범위를 읽어 거래처 목록 생성
- 거래처명 일부 검색 후 선택 가능
- F열 값에서 공백을 무시하고 `반값 x1.4` 문구 포함 여부 판정
- 계산 가능 거래처만 계산창 노출
- 자동 보정값이 필요할 때 빨간 글씨로 표시
- 보정이 필요한 경우에만 상품가(C) 직접 수정 가능
- 최근 계산 결과 5개까지 저장해서 다시 확인 가능

## 엑셀 파일 사용 방식

앱 폴더 안에 아래 둘 중 하나의 파일명이 있으면 자동으로 읽습니다.

- `업무리스트.xlsx`
- `업무리스트 (1).xlsx`

시트가 많거나 파일 용량이 커도, 앱은 `거래처DB` 시트의 필요한 셀 범위만 읽도록 구성되어 있습니다.

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud / GitHub 배포

1. 이 폴더 전체를 GitHub 저장소에 업로드
2. Streamlit Cloud에서 저장소 연결
3. 메인 파일을 `app.py`로 지정
4. 배포 실행

## 계산식

- A = x / 2 → 매입가
- B = (x / 2) × 1.4 → 1.4배
- C = B를 10,000원 단위로 올림 → 상품가
- D = A - C → 차액
- E = C × 1.1 → 판매가

추가 규칙:

- `30000 - D`가 양수이면 그 값을 C에 자동으로 더함
- 이 경우 C는 수정 가능
- C 수정 시 D와 E는 실시간으로 다시 계산됨
- `30000 - D`가 0 이하이면 C는 수정 불가
