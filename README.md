# 브랜드별 단가 계산기

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 포함 기능
- 거래처 일부 검색 후 선택
- `거래처DB` 시트의 `A3:A63`, `F3:F63` 기준 읽기
- 다음 계산식 자동 분기
  - `반값 x1.4`
  - `국내제작 x1.5`
  - `국내제작 x1.5 + 4만원`
  - `프로젝트가 x1.5 + 2만원`
- 최근 계산 결과 5개 보기

## 파일 구성
- `app.py`
- `requirements.txt`
- `README.md`
- `업무리스트.xlsx`

## 참고
- 시트가 많아도 필요한 시트와 열만 읽도록 구성했습니다.
- `반값 x1.4`는 차액이 30,000원 미만이면 자동 보정이 들어갑니다.
